# rate-limit-reducer

A lib to help reduce your app API keys from being rate-limited when processing big data

# Reducer Vector

app_name
api_key_data
rate_limit_alloc
rate_limit_left
rate_limit_reset_ts

### Unit Tests

`cargo run --workspace -- --nocapture`

### Run the program

`cargo run`

### Use clippy to lint and clean rust code

`cargo clippy --fix --allow-dirty`
